Modelling and Model Transformation

In order to model and enable the verification of your models, you should install both the Eclipse plugin for TiVA DSL and the TiVA Core tool.

Eclipse TiVA DSL plugin
to edit models in the TiVA DSL and verify the syntax
-- available as an Eclipse plugin

TiVA Core tool
to transform models into networks of timed automata in the XTA format
-- available both as an executable file jar and as java source code

Model Verification

Timed verifications of the models (once transformed into networks of timed automata) are available in TiVA Core tool, since the UPPAAL Engine is encapsulated with TiVA Core tool. Nevertheless, to have counter examples, or to make graphical simulations, one can use UPPAAL Tool


UPPAAL tool
an integrated tool environment with verification features for real-time systems
-- available as binaries (easy install)
